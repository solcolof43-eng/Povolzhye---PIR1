4. Цели и предпосылки закупок

6 категорий
Обустройство месторождений и реконструкция

Крупнейшая категория. Зрелые провинции ПФО требуют постоянной актуализации ПД: новые кусты скважин, нефтепроводы, точки врезки. 8 месторождений Белкамнефти, НГДУ Татнефти, Ардалинское НГП.

Розничная сеть АЗС Татнефти

Стратегическое направление холдинга. 7 процедур: долгосрочные договоры на ПИР по группам АЗС, согласования с ГАИ РФ и ГК «Автодор», модернизация существующих объектов.

Нормативно обязательные проекты

Требования МЧС (ОПС/СОУЭ), Росприроднадзора (СЗЗ, рекультивация), Ростехнадзора (декларации ОПО), Рослесхоза (лесные участки). Стабильный регулярный спрос.

Обучение и аттестация ГИП (Роснефть)

Централизованная программа подготовки сметной службы и ГИП-корпуса: ГРАНД-Смета, ценообразование в строительстве, профессиональные экзамены 7-го уровня НАРК.

Техническое перевооружение оборудования

Модернизация котлоагрегатов, технологических печей ТАНЕКО, установка производства ПЭТ (АО «Экопэт», Калининград), новые химические производства Татнефти.

Сметное и проектное ПО

Роснефть стандартизировала три продукта: ГРАНД-Смета + Адепт:Проект + CADLib. Регулярные закупки обновлений и лицензий. Лукойл использует собственную ЭТП с НМЦК «1 руб.».

Ключевые предпосылки и драйверы спроса

Первичный драйвер

Поддержание и модернизация зрелой добычи Волго-Уральской провинции. Татарстан, Башкортостан, Удмуртия, Самарская и Оренбургская области — старейшие российские нефтеносные регионы с постоянной потребностью в актуализации ПД.

Регуляторные требования

123-ФЗ (пожарная безопасность), 7-ФЗ (охрана окружающей среды), 116-ФЗ (промышленная безопасность ОПО), Лесной кодекс РФ, Закон «О недрах» — формируют обязательный нормативный спрос вне инвестиционного цикла.

Розничный бизнес

Долгосрочное развитие сети АЗС Татнефти создаёт устойчивый годовой портфель ПИР независимо от добычного цикла. Договоры покрывают группы объектов одновременно.

Паттерн «длинного хвоста»

Позиции с низким спросом — узкоспециализированные работы с высокими лицензионными барьерами (ФСТЭК, Росатом, Ростехнадзор, Росаккредитация) при небольшом годовом объёме. Конкуренция в них ограничена кругом аккредитованных исполнителей, но и рыночный потенциал невелик. Для новых участников эти ниши малопривлекательны из-за стоимости получения допусков при единичном спросе.

8. Барьеры входа и уникальность рынка

Специфика ПФО

СРО проектировщиков

Обязательное членство в саморегулируемой организации проектировщиков и изыскателей по Градостроительному кодексу РФ — базовый барьер для любых ПИР по объектам капстроительства.

Лицензии Ростехнадзора

Проектирование ОПО, проведение экспертизы промышленной безопасности, маркшейдерские работы на горных отводах — требуются для большинства лотов по обустройству месторождений и техперевооружению.

Лицензия Роспотребнадзора/Росатома

Работа с источниками ионизирующего излучения — требуется для комплексных инженерных изысканий с радиационным обследованием объектов (Татнефть, тендер 4364962).

Лицензия ФСТЭК России

Техническая защита конфиденциальной информации — для проектирования систем ИБ объектов критической информационной инфраструктуры НПЗ АО «ТАНЕКО» (Татнефть, 4360818).

Аккредитация Росаккредитации

Проведение негосударственной экспертизы проектной документации (ГПН-Оренбург). Требует отдельного прохождения аккредитационной процедуры.

Кадастровые инженеры

Аттестованные кадастровые инженеры — члены СРО КИ — для процедур межевания земель ГПН-Оренбург и работы с лесными участками Белкамнефти.

Согласования с ГАИ и ГК «Автодор»

Уникальная черта АЗС-сегмента Татнефти: опыт согласований с нетипичными для ПИР по нефтедобыче регуляторами — обязательное требование в 2+ процедурах.

Аттестация ГИП (НАРК)

Профессиональный экзамен 7-го уровня «Главный инженер проекта» — корпоративное требование Роснефти для ключевых специалистов. Роснефть регулярно проводит аттестационные тендеры.

Корпоративные стандарты ПО

Роснефть: ГРАНД-Смета + Адепт:Проект + CADLib. Лукойл: собственная ЭТП. Татнефть: закрытая система «Запрос предложений». Несовместимость стандартов усиливает привязанность к конкретным поставщикам.
